🖼️ Picture Project  

A web application for uploading and previewing images with drag & drop functionality.  
Built using JavaScript and Gulp for asset bundling.

👉 Live Demo: https://itssonyamisyura.github.io/Picture_project/  

👉 Repository: https://github.com/itssonyamisyura/Picture_project  

🚀 About  

This project is a frontend application that allows users to upload images using drag & drop and preview them instantly.  

The app demonstrates working with file uploads, DOM manipulation, and build tools like Gulp.  

✨ Features  
📂 Drag & Drop image upload  
🖼️ Image preview after upload  
📁 File input support  
⚡ Fast bundling with Webpack via Gulp  
📱 Responsive layout  

🛠️ Technologies Used  

JavaScript (ES6+)  
HTML5  
CSS / SCSS  
Gulp (task runner)  
Webpack (bundling inside Gulp)  
Babel (ES6 transpilation)

🖼️ Picture Project  

A web application for uploading and previewing images with drag & drop functionality.  
Built using JavaScript and Gulp for asset bundling.

👉 Live Demo: https://itssonyamisyura.github.io/Picture_project/  

👉 Repository: https://github.com/itssonyamisyura/Picture_project  

🚀 About  

A web application for working with images, including drag & drop uploads, form handling, and interactive UI components.

The app combines multiple features such as uploading files, handling forms, and dynamic UI behavior.

The app demonstrates:

- Drag & Drop file uploading  
- Working with forms and sending data  
- Handling user interactions  
- Modular JavaScript architecture  
- Dynamic UI updates  

---

## ✨ Features

- 🖼️ Drag & Drop image upload  
- 📂 File upload via input  
- 📝 Form submission handling  
- 📡 Sending data 
- ⚡ Dynamic UI updates  
- 📱 Responsive layout  

🛠️ Technologies Used  

JavaScript (ES6+)  
HTML5  
CSS / SCSS  
Gulp (task runner)  
Webpack (bundling inside Gulp)  
Babel (ES6 transpilation)
